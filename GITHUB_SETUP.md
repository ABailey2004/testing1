# Blue Light RPC Website - GitHub Repository

## Quick Setup Instructions

### 1. Repository Setup
1. Create a new GitHub repository named `testing1`
2. Upload all files from this folder to the repository
3. Make sure the repository is public (required for GitHub Pages free tier)

### 2. GitHub Pages Configuration
1. Go to your repository Settings tab
2. Scroll to "Pages" section in the left sidebar
3. Under "Source", select "Deploy from a branch"
4. Choose "main" branch and "/ (root)" folder
5. Click Save

### 3. Custom Domain (Optional)
If you have your own domain:
1. Edit the `CNAME` file and replace the comment with your domain
2. In your domain provider's DNS settings, add:
   - A records pointing to GitHub Pages IPs (see README.md)
   - CNAME record for www subdomain
3. In GitHub Pages settings, enter your custom domain and enable HTTPS
